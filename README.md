# DSK Architecture Studio

  A digital architectural gallery that uses "negative space" (white) to highlight structural depth (dark grey) and focal points (muted mustard yellow).

  ## Design System

  - **Background**: Pure White (#FFFFFF)
  - **Structure**: Anthracite / Dark Charcoal (#2F2F2F)
  - **Accent**: Muted Mustard (#F4D03F) — thin lines, borders, bullet points

  ## Features

  - Floating hero with "DSK" title overlapping a dark-framed image
  - Broken grid project gallery with category filtering
  - Project detail modals
  - Contact form with database persistence
  - Responsive design with mobile navigation

  ## Tech Stack

  - **Frontend**: React, Vite, Tailwind CSS, TanStack Query
  - **Backend**: Express.js, PostgreSQL, Drizzle ORM
  - **Fonts**: Space Grotesk, Playfair Display, JetBrains Mono

  ## Getting Started

  ```bash
  npm install
  npm run db:push
  npm run dev
  ```
